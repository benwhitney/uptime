Internet Outage Report

About this report

This report was created using Uptime Home Internet Reliability Monitor (https://github.com/benwhitney/uptime). Running on a Node.js server, this app uses is-online to check online status, by performing the following checks in parallel:

  • Retrieve icanhazip.com via HTTPS
  • Query myip.opendns.com on OpenDNS
  • Retrieve Apple's Captive Portal test page
If any of the above succeed, within 5 seconds, the internet is online. Online status is checked every {{ updateIntervalSeconds }} seconds and logged.
